# LSC LiDAR 테스트 프로그램

LSC 라이다 센서의 실시간 데이터를 웹으로 시각화하는 프로그램입니다.

## � 빠른 시작

### 1. 설치
```bash
npm install
```

### 2. 실행
```bash
npx ts-node src/server.ts
```

### 3. 접속
브라우저에서 `http://localhost:3000` 접속

## 🔧 사용법

1. **연결**: IP 주소(`192.168.0.10`)와 포트(`8000`) 입력 후 Connect 클릭
2. **시각화**: 라이다 데이터가 실시간으로 원형으로 표시됨
3. **제어**: 
   - 슬라이더로 회전 각도 조절
   - 마우스 휠로 확대/축소
   - 마우스로 거리/각도 확인

## � 요구사항

- Node.js 16.0+
- LSC 라이다 센서 (네트워크 연결)

## �️ 문제 해결

**연결 안될 때:**
- IP 주소와 포트 확인
- 라이다 전원 및 네트워크 연결 확인
- 방화벽 확인 (포트 3000, 8000)

**의존성 문제:**
```bash
rm -rf node_modules package-lock.json
npm install
```

# LSC LiDAR 테스트 프로그램
LSC 라이다 센서의 실시간 데이터를 웹으로 시각화하는 프로그램입니다.

## � 빠른 시작

### 1. 설치
```bash
npm install
```
### 2. 실행
```bash
npx ts-node src/server.ts
```

### 3. 접속
브라우저에서 `http://localhost:3000` 접속

## 🔧 사용법

1. **연결**: IP 주소(`192.168.0.10`)와 포트(`8000`) 입력 후 Connect 클릭
2. **시각화**: 라이다 데이터가 실시간으로 원형으로 표시됨
3. **제어**: 
   - 슬라이더로 회전 각도 조절
   - 마우스 휠로 확대/축소
   - 마우스로 거리/각도 확인

## ⚙️ 초기화 모드 설정

`src/server.ts` 파일의 413번 라인에서 초기화 모드를 선택할 수 있습니다:

```typescript
const USE_FULL_INITIALIZATION = false;  // 최소 모드 (기본값)
// const USE_FULL_INITIALIZATION = true;   // 전체 모드
```

### **최소 모드** (`false`)
- **속도**: 약 0.7초
- **단계**: 연결 대기 → 로그인 → 스캔 시작
- **용도**: 빠른 테스트, 기본 설정

### **전체 모드** (`true`)
- **속도**: 약 1.7초  
- **단계**: 연결 대기 → 센서 정보 → 로그인 → 설정 읽기 → 설정 쓰기 → 스캔 시작
- **용도**: 정밀 설정, 각도 범위 지정 (-45° ~ 225°)

**설정 변경 후 서버를 재시작해야 합니다.**

## � 요구사항

- Node.js 16.0+
- LSC 라이다 센서 (네트워크 연결)

## �️ 문제 해결

**연결 안될 때:**
- IP 주소와 포트 확인
- 라이다 전원 및 네트워크 연결 확인
- 방화벽 확인 (포트 3000, 8000)

**의존성 문제:**
```bash
rm -rf node_modules package-lock.json
npm install
```
